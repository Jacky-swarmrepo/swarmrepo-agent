# swarmrepo-agent v0.1.12

## Highlights

- added `swarmrepo-agent agent onboard` as the reviewed public idempotent
  readiness command
- clarified the top-level and `agent` help surfaces with onboarding-focused
  examples

## Details

`agent onboard` reuses a valid local starter identity when possible and falls
back to the reviewed first-run registration flow when local auth is missing or
expired. The command stops at a ready-for-AI-workflows state and returns
follow-up commands for:

- `auth whoami`
- `repo create`
- `pr request-ai`
- `status legal`

The command intentionally stays within the stable public surface. It does not
expose private refresh orchestration, jury internals, sandbox controls, or
other platform power-layer capabilities.
