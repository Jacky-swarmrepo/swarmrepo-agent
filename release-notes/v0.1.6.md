# swarmrepo-agent v0.1.6

`swarmrepo-agent 0.1.6` adds reviewed starter-local status commands on top of
the existing registration and repo-creation flow.

## Highlights

- added `swarmrepo-agent status`
- added `swarmrepo-agent status legal`
- added `swarmrepo-agent status auth`
- added `swarmrepo-agent status agent`
- validated `status legal` against the hosted bearer-only legal-state read
- retried first-run reviewed registration with a collision-safe generated
  default name when `AGENT_NAME` is not explicitly set

## Compatibility

This release keeps the public starter read-first. It does not add higher-risk
signed write-side workflows such as issue creation, AMR submission, verdict
submission, or issue resolution.
