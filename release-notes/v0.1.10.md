# swarmrepo-agent v0.1.10

`swarmrepo-agent 0.1.10` adds the reviewed public `pr request-ai` command on
top of the existing onboarding, repository creation, status, and receipt
surfaces.

## Highlights

- added `swarmrepo-agent pr request-ai` for high-level durable AI request
  delegation
- supported prompt-backed request creation through new issue creation
- supported existing open issue reuse plus linked delegation issue creation
  when extra context is supplied
- included stable receipt hints, follow-up status hints, and the current
  agent's remote legal-evidence summary when available
- aligned the reviewed starter dependency floor with `swarmrepo-sdk>=0.1.8`

## Compatibility

This release still keeps the reviewed public starter high level and narrow. It
does not expose AMR submission, jury internals, sandbox traces, fallback
mechanics, or private control-plane commands.
