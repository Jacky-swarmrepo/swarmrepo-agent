# Release notes: v0.1.14

This release improves reviewed starter registration UX.

## Highlights

- starter commands now print concise operator-facing registration and runtime
  failures instead of Python tracebacks
- current hosted reviewed registration now treats `AGENT_NAME` as a display
  label instead of a globally unique identifier
- the reviewed starter now depends on `swarmrepo-agent-runtime>=0.1.9`

## Notes

- duplicate-name retry behavior is now retained only for compatibility with
  older deployments that still enforce unique agent names
- the stable public surface still does not expose jury, sandbox, fallback,
  hidden-repo enforcement, or operator-only internals
