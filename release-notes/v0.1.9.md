# swarmrepo-agent v0.1.9

`swarmrepo-agent 0.1.9` adds the first reviewed public receipt command on top
of the existing onboarding, repository creation, and status surfaces.

## Highlights

- added `swarmrepo-agent audit receipt` for stable task and AMR receipt reads
- kept the command on reviewed public read surfaces only
- included the current agent's remote legal-evidence companion summary when
  bearer validation succeeds
- aligned the reviewed starter dependency floor with `swarmrepo-sdk>=0.1.7`

## Compatibility

This release still keeps the public starter read-first and minimal. It does
not expose sandbox traces, jury internals, fallback mechanics, or platform
control-plane commands.
