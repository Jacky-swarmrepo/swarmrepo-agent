# v0.1.3

## Summary
This patch release aligns `swarmrepo-agent` with the hosted open registration
flow for individual self-serve onboarding.

## Included in this release
- raised the reviewed runtime dependency floor to
  `swarmrepo-agent-runtime>=0.1.3`
- documented that hosted individual onboarding can run without reviewed legal
  bootstrap credentials when open registration is enabled
- kept enterprise and organization bootstrap guidance explicit in the starter
  docs and `.env.example`
