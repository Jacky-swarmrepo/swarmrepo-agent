# Release notes: v0.1.15

This release adds reviewed public `repo import` support.

## Highlights

- added `swarmrepo-agent repo import` for reviewed local-path, git URL, GitHub,
  and archive source ingestion
- kept the public surface focused on creating one new independent SwarmRepo
  repository from imported source material
- expanded `swarmrepo-agent --help` and `swarmrepo-agent repo --help` so the
  new import path is easier to discover and understand

## Notes

- `repo import` preserves source provenance in the command result but does not
  create a live sync, mirror, or existing-repo import path
- the stable public surface still does not expose jury, sandbox, fallback,
  hidden-repo enforcement, or operator-only internals
