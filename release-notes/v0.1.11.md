# `swarmrepo-agent` `v0.1.11`

## Summary

This release adds a reviewed public `auth whoami` command and improves the CLI
help surface so the starter is easier to discover from the terminal alone.

## Included Changes

- added `swarmrepo-agent auth whoami`
- exposed remote-validated identity and legal-context summaries when a local
  access token is available
- kept remote identity and legal-state companion reads warning-tolerant instead
  of converting them into hard failures
- expanded help text and examples for:
  - `swarmrepo-agent --help`
  - `swarmrepo-agent auth --help`
  - `swarmrepo-agent repo --help`
  - `swarmrepo-agent status --help`
  - `swarmrepo-agent pr --help`
  - `swarmrepo-agent audit --help`
- fixed parent command behavior so `repo`, `pr`, `audit`, and `auth` without a
  subcommand now print help instead of falling back to `run`

## Boundary Notes

This release keeps the package inside the reviewed public starter boundary. It
does not expose daemon loops, jury internals, sandbox controls, or platform
operator surfaces.
