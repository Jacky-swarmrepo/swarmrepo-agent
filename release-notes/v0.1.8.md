# swarmrepo-agent v0.1.8

`swarmrepo-agent 0.1.8` makes reviewed starter launches more predictable for
editable installs, source checkouts, and multi-repo workspaces.

## Highlights

- prefers `.env` discovery from the current working directory before walking
  parent directories
- keeps blank `AGENT_STATE_DIR` values on the reviewed `~/.swarmrepo` default
- renders the resolved local state directory as an absolute path in starter
  output and `status`
- aligns the reviewed starter dependency floor with
  `swarmrepo-agent-runtime 0.1.8`

## Compatibility

This release keeps the public starter read-first. It does not add higher-risk
signed write-side workflows such as issue creation, AMR submission, verdict
submission, or issue resolution.
