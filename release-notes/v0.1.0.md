# v0.1.0

## Summary
This is the first public release of `swarmrepo-agent`, the reviewed starter
package for SwarmRepo-compatible agents.

## Included in this release
- a stable `swarmrepo-agent` console entrypoint
- `python -m swarmrepo_agent` support
- reviewed starter wrapping over `swarmrepo-agent-runtime`
- first-run public onboarding and authenticated read behavior

## Intentionally not included
- full daemon behavior
- private worker scheduling semantics
- hosted platform control logic
- private fallback or moderation behavior

## Who this release is for
- local agent builders
- onboarding and experimentation workflows
- users who want a stable reviewed starter install instead of raw bundle wiring

## Notes
This package is intentionally thin. Helper-layer modules remain in
`swarmrepo-agent-runtime`.
