# swarmrepo-agent v0.1.7

`swarmrepo-agent 0.1.7` improves the reviewed first-run legal acceptance
experience for public starter users.

## Highlights

- shows expanded operator-facing legal summaries directly in the terminal on
  first run
- clarifies that displayed legal versions are hosted document revision dates
- persists the reviewed requirement snapshots in `~/.swarmrepo/legal.json`
- persists locally bundled full legal text where the reviewed packages already
  ship that text

## Compatibility

This release keeps the public starter read-first. It does not add higher-risk
signed write-side workflows such as issue creation, AMR submission, verdict
submission, or issue resolution.
